# Gestão de Projetos L&M

Sistema de Gestão Integrado para controle de empresas, projetos e tarefas, focado em processos contábeis e consultoria empresarial.

## 🚀 Funcionalidades

- **Dashboard de Empresas**: Visualização detalhada de clientes, incluindo dados cadastrais, regimes tributários e status de projetos.
- **Gestão de Projetos**: Acompanhamento de fases, objetivos e cronogramas.
- **Controle de Tarefas**: Gerenciamento granular de atividades com status em tempo real.
- **Busca Automática de CNPJ**: Integração para preenchimento automático de dados cadastrais via CNPJ.
- **Controle de Acesso (RBAC)**: Diferentes níveis de permissão (Admin, Manager, Viewer).
- **Geração de Relatórios PDF**: Exportação de dados do projeto e tarefas em formato PDF profissional.
- **Interface Responsiva**: Design moderno e otimizado para diferentes dispositivos utilizando Tailwind CSS.

## 🛠️ Tecnologias Utilizadas

- **Frontend**: React, TypeScript, Vite.
- **Estilização**: Tailwind CSS, Lucide React (ícones).
- **Backend/Banco de Dados**: Supabase (PostgreSQL).
- **Animações**: Framer Motion.

## 📋 Pré-requisitos

- Node.js (versão 18 ou superior)
- NPM ou Yarn

## 🔧 Instalação e Execução

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Configure as variáveis de ambiente no arquivo `.env` (baseado no `.env.example`).
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
